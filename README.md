# 装修需求记录

这是一个用于整理室内装修需求的 Obsidian 知识库。

## 项目结构

```
装修需求草案/       # 整理后的需求文档
  初步需求.md       # 初步需求汇总
  家具/             # 家具相关需求
  家电/             # 家电相关需求
  生活用品/         # 生活用品清单
  src/              # 图片资源
草稿/               # 草稿与记录
关系图.canvas        # Obsidian 关系图
Agent.md            # AI 协作上下文说明
```

## 使用方式

使用 [Obsidian](https://obsidian.md/) 打开此文件夹作为 Vault。

## 说明

当前阶段为日常需求记录阶段，通过 diary study 的方式记录每天的生活场景、行为动线、物品使用、收纳需求等，为后续室内设计提供依据。
